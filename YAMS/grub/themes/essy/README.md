[GRUB2] Essy theme
===========

Essy is a GRUB2 theme for YAMS.

This theme is based on	
 - [Dacha204`s Ettery GRUB2 theme](https://github.com/Dacha204/grub2-themes-Ettery)

Supported and tested resolutions:
	3840x2160		1360x768
	1920x1080		1280x800
	1600x1200		1152x864
	1776x1000		1280x768
	1680x1050		1280x720
	1400x1050		1024x768
	1600x900		1152x648
	1280x1024		800x600
	1440x900		720x480
	1280x960		640x480
